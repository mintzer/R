פרק 2 לולאות

איטרציה (iteration) היא פעולה שמתארת מעבר על אובייקט שמכיל מספר איברים, איבר אחרי איבר. בהתחלה האיבר הראשון, השני, השלישי וכן הלאה.

2.1 לולאת for

דוגמא ללולאה שעוברת על וקטור של מספרים מ1 עד 10 ולכן רצה בדיוק עשר פעמים:

for (i in 1:10) {
  print(i)
}

## [1] 2
## [1] 3
## [1] 4
## [1] 5
## [1] 6
## [1] 7
## [1] 8
## [1] 9
## [1] 10

לולאת for שעושה איטרציה על וקטור של משתנה מסוג “character”

animals <- c('lion','monkey','tiger')
for (a in animals) {
  print(a)
}  
## [1] "lion"
## [1] "monkey"
## [1] "tiger"
# using i as index
animals <- c('lion','monkey','tiger')
fruits <- c('apple','orange','pineapple')
for (i in 1:length(animals)) {
  print(paste(i, animals[i], fruits[i]))
}
## [1] "1 lion apple"
## [1] "2 monkey orange"
## [1] "3 tiger pineapple"

2.2 לולאת while

i <- 1
while (i < 5) {
  print (i)
  i <- i + 1
}
## [1] 1
## [1] 2
## [1] 3
## [1] 4
index <- 1
while (index < 7) {
  if (index %% 2 == 0) {
    print(paste('the number',index, 'is even'))
  }
  else if (index == 5) {
    print('5!')
  }
  else if (index %% 2 != 0) {
    print(paste('the number',index, 'is not even and not 5'))
  }
  index <- index + 1
}
## [1] "the number 1 is not even and not 5"
## [1] "the number 2 is even"
## [1] "the number 3 is not even and not 5"
## [1] "the number 4 is even"
## [1] "5!"
## [1] "the number 6 is even"
