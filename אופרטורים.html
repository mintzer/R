פרק 3 - אופרטורים

3.1 אופרטורים אריתמטיים

אופרטור תיאור
+ חיבור
- חיסור
* כפל
/ חילוק
^ העלאה בחזקה
%% מודולו - שארית החלוקה
%/% חלוקה ללא שארית

3.2 אופרטורים לוגיים

אופרטור פעולה דוגמה הסבר
& AND expr1 & expr2 מחזיר ערך TRUE כאשר expr1 ו- expr2 הם TRUE
| OR expr1 | expr2 מחזיר ערך TRUE כאשר אחד מהערכים, expr1 או expr2, הוא TRUE
! NOT expr! מחזיר ערך TRUE כאשר expr אינו TRUE
%in% a %in% b מחזיר TRUE כאשר a מוכל בוקטור b

3.3 אופרטורים של השוואה

אופרטור דוגמה הסבר
> value1 < value2 מחזיר ערך TRUE אם הערך הראשון נמוך מהערך השני
=> value1 <= value2 מחזיר ערך TRUE אם הערך הראשון קטן מהערך השני או שווה לו
< value1 > value2 מחזיר ערך TRUE אם הערך הראשון גדול מהערך השני
=< value1 >= value2 מחזיר ערך TRUE אם הערך הראשון גדול מהערך השני או שווה לו
== value1 == value2 מחזיר ערך TRUE אם הערך הראשון שווה לערך השני
=! value1 != value2 מחזיר ערך TRUE אם הערך הראשון אינו שווה לערך השני
